TAMSAFE APP DESIGN

This Project was part of our HCI or Human Computer Interaction subject during the second semester. We were tasked to create and design a prototype. but this desing was only made entirely using Canva. Among all the Projects I've worked on, this one is one of my personal favorites because I dedicated so much time and effort in creating and designing it. The concept of this Project is a Campus Safety Map designed exclusively for FEU students employees, and professors. The app features and emergency alarm system that users can activate by pressing an alert button whenever they're in danger or need immediate assistance. This idea was inspired by apps like Life 360 and Google Maps, but it's tailored specifically for the FEU community. if you're as Tamaraw, this app is made for you..
